    Problem Description

    Name: Boids Simulation

    Problem Statement: Boids is an artificial intelligence so they will making decision depend on this 3 rules are cohesion, alignment, and separation.

    Problem Specification: Calculate average distance of boids for cohesion. Calculate average velocity of boids exxcept itself for alignment. Calculate distance between the boids for seperation.

    User Interface Information

    Name: Cohesion Description: Boids center of mass. The average distance of all the boid

    Name: Alignment Description: The average of boids velocity

    Name: Seperation Description: A distance that boids will maintain from each others

    Name: Exit Description: Quit the aplication

    Name: Resest Sliders Description: reset Cohension, Alignment, and Seperation back to their default value

    Name: Target Description: when the check box is check all the boids will move toward the target
